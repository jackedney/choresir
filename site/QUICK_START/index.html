Quick Start Guide

Get the bot running locally and test your first message.

Time: 30 minutes Prerequisites: Twilio account with WhatsApp enabled, OpenRouter API key

1. Submit WhatsApp Templates (Start Now)

Approval takes 1-2 days. Submit while doing other setup.

Required templates: chore_reminder, verification_request, conflict_notification

See the Setup Guide for detailed template specifications.

2. Setup Local Environment

# Install dependencies
uv sync

# Download PocketBase
task setup

# Configure environment
cp .env.example .env
nano .env  # Add your tokens

Required .env variables: 

POCKETBASE_URL=http://127.0.0.1:8090
POCKETBASE_ADMIN_EMAIL=admin@example.com
POCKETBASE_ADMIN_PASSWORD=your_admin_password_here
OPENROUTER_API_KEY=<your key>
TWILIO_ACCOUNT_SID=<your sid>
TWILIO_AUTH_TOKEN=<your token>
TWILIO_WHATSAPP_NUMBER=<your number>
HOUSE_CODE=HOUSE123
HOUSE_PASSWORD=SecretPass

Optional: 

LOGFIRE_TOKEN=<your token>  # Get from https://logfire.pydantic.dev

3. Setup ngrok

brew install ngrok
ngrok config add-authtoken YOUR_TOKEN  # Get token from https://ngrok.com after signing up

4. Start Services

Terminal 1: PocketBase 

./pocketbase serve
First time: Create admin account at http://127.0.0.1:8090/_/

Terminal 2: FastAPI 

uv run fastapi dev src/main.py --port 8000

Terminal 3: ngrok 

ngrok http 8000
Copy the HTTPS URL (e.g., https://abc123.ngrok-free.app)

5. Configure Twilio WhatsApp Webhook

  1. Go to Twilio Console
  2. Navigate to Messaging → WhatsApp Sandbox → Sandbox Configuration
  3. Set webhook configuration:
  4. When a message comes in: https://abc123.ngrok-free.app/webhook
  5. HTTP Method: POST
  6. Save configuration

6. Create First Admin User

Send to bot: 

Join HOUSE123 SecretPass YourName

Then in PocketBase admin (http://127.0.0.1:8090/_/): 1. Go to Collections → users 2. Edit your user: set role to "admin", status to "active" 3. Save

7. Test

Basic message: 

Hello

Create chore: 

Remind me to water plants every 3 days

Log completion: 

I watered the plants

Troubleshooting

Webhook verification fails: - FastAPI running on port 8000? - ngrok tunnel active? - Webhook URL correctly configured in Twilio Console?

Bot doesn't reply: - OpenRouter API key valid? - Twilio credentials (SID and auth token) valid? - Check FastAPI logs for errors
