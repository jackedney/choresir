### US-003: Create Getting Started documentation
Status: in_progress
Depends on: US-002

Description:
As a new user, I want clear getting started guides so I can quickly set up and use the WhatsApp bot.

Acceptance Criteria:
- [ ] Write installation guide (prerequisites, commands, verification)
- [ ] Write configuration guide (environment variables, PocketBase setup)
- [ ] Write first run guide (sending first message, verifying connection)
- [ ] Include code examples with syntax highlighting
- [ ] Example: Installation includes uv install, PocketBase download, environment setup
- [ ] Negative case: Missing prerequisite causes failure in setup -> document error message and solution
- [ ] Verify all commands are accurate and tested
