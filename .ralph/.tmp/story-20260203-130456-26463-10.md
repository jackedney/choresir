### US-010: Add spelling and lint checks
Status: in_progress
Depends on: US-001

Description:
As a contributor, I want automated spelling and lint checks so documentation quality is maintained.

Acceptance Criteria:
- [ ] Install markdownlint package
- [ ] Install textlint package
- [ ] Configure markdownlint rules for Markdown files
- [ ] Configure textlint rules for spelling and grammar
- [ ] Add quality gate commands to pyproject.toml or similar
- [ ] Example: 'uv run markdownlint docs/' catches formatting issues
- [ ] Negative case: Typo in documentation -> textlint catches spelling error
- [ ] Document quality gate commands in Contributors guide
