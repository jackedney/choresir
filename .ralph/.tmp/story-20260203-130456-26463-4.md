### US-004: Create User Guide documentation
Status: in_progress
Depends on: US-002

Description:
As an end user, I want comprehensive user guides so I can understand all features and capabilities.

Acceptance Criteria:
- [ ] Document all available features and commands
- [ ] Provide usage examples for each feature
- [ ] Include common use cases and workflows
- [ ] Add FAQ section for common questions
- [ ] Example: 'How to create a new chore' with step-by-step instructions
- [ ] Negative case: User tries unsupported command -> document error handling
- [ ] Ensure language is accessible to non-technical users
