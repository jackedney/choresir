### US-012: Create README with documentation link
Status: in_progress
Depends on: US-011

Description:
As a new user or contributor, I want clear direction to the documentation so I can find what I need.

Acceptance Criteria:
- [ ] Update project README with clear link to documentation
- [ ] Add instructions for building documentation locally
- [ ] Include quick links to key sections (Getting Started, Contributors)
- [ ] Example: 'For full documentation, run `mkdocs serve` and visit http://127.0.0.1:8000'
- [ ] Negative case: Outdated link -> broken link in README
- [ ] Keep README concise with reference to full docs
