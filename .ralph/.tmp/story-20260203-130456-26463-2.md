### US-002: Configure documentation navigation structure
Status: in_progress
Depends on: US-001

Description:
As a contributor, I want clear navigation structure so I can find documentation easily.

Acceptance Criteria:
- [ ] Define nav structure in mkdocs.yml with all main sections
- [ ] Create directory structure: docs/getting-started/, docs/user-guide/, docs/contributors/, docs/architecture/, docs/agents/, docs/adr/
- [ ] Add section index files (README.md or similar)
- [ ] Example: 'Getting Started' section has subpages for Installation, Configuration, First Run
- [ ] Negative case: Broken navigation link -> 404 error in build
- [ ] Verify navigation renders correctly in sidebar
