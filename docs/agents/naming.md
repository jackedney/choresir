# Naming Conventions

This page describes naming conventions for Pydantic AI agents in WhatsApp Home Boss.

## Agents

Use snake case (e.g., `household_manager`).

## Tools

Use snake case, prefixed with `tool_` (e.g., `tool_log_chore`).

## Models

Use PascalCase Pydantic models (e.g., `LogChoreParams`).
